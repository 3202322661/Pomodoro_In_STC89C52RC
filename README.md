# 番茄钟 (Pomodoro Timer) — STC89C52RC

基于 STC89C52RC 单片机的番茄工作法计时器，使用 Keil C51 (uVision5) 编译环境。
采用 4 位共阴数码管显示，3 键操作，工作时长与休息时长可独立调节 (1~60 分钟)。

---

## 功能特性

- 4 位共阴数码管实时显示 MM:SS 倒计时 (小数点自动闪烁分隔)
- 3 键操作：设置 (切换模式)、启停 (启动/暂停)、递加 (调整时长)
- 工作时长与休息时长独立可调，范围 1~60 分钟 (循环递增)
- Timer0 1ms 中断同时驱动数码管动态扫描与时基生成
- 设置模式下对应数字位以 0.5s 周期闪烁指示当前调节目标
- 倒计时归零时蜂鸣器发出 5 声短促告警音并自动切换阶段
- **深度休眠省电**：空闲/暂停 5 分钟无操作自动进入 PCON 掉电模式 (待机 < 0.1μA)，INT0/INT1 一键唤醒

## 硬件连接

```
STC89C52RC          外设
──────────         ──────────
P0.0~P0.7  ────   数码管 a~g + dp (段码, 共阴)
P2.0~P2.3  ────   数码管位选 0~3   (低电平有效)
P3.2       ────   按键 KEY_SET     (设置键, 低电平有效)
P3.3       ────   按键 KEY_START   (启停键, 低电平有效)
P3.4       ────   按键 KEY_ADD     (递加键, 低电平有效)
P1.5       ────   有源蜂鸣器       (三极管驱动)
```

> 数码管段码线与 P0 口之间建议串联 220Ω 限流电阻。P0 口需外接 10kΩ 上拉电阻排。
> 位选线 P2.0~P2.3 需通过三极管 (如 9012 PNP) 扩流驱动，每个位选需串联基极限流电阻。

## 编译与烧录

### Keil 环境配置

一、用 Keil uVision5 打开 `Pomodoro.uvproj`

二、确认 Device 为 **STC89C52RC**（若 Keil 未安装 STC 器件包，可改用 **AT89C52** 作为兼容替代）

三、确认晶振频率设置为 **11.0592 MHz**

四、编译选项：Memory Model: **Small**, Optimization: **Level 8 (Speed)**

五、编译成功后生成 `Pomodoro.hex`，通过 **STC-ISP** 烧录

## 定时器计算

```
晶振:         11.0592 MHz
机器周期:      12 / 11.0592 MHz ≈ 1.085 μs (12T 模式)
目标中断:     1 ms
所需计数值:   1000 / 1.085 ≈ 922
16-bit 初值:  65536 - 922 = 64614 = 0xFC66
TH0 = 0xFC, TL0 = 0x66

时基生成 (ISR 内):
  ms_count 每 1ms 递增
  ms_count == 500  → half_sec_flag 翻转 (0.5s 闪烁驱动)
  ms_count >= 1000 → sec_flag = 1, ms_count 归零 (1s 倒计时驱动)
```

## 使用说明

### 按键操作

按下 **KEY_SET** (P3.2) 在三种主要模式间循环切换：

- 空闲/运行/暂停 → 设置工作时长 (分钟位闪烁)
- 设置工作时长 → 设置休息时长 (秒位闪烁)
- 设置休息时长 → 返回空闲

按下 **KEY_START_STOP** (P3.3) 控制运行与暂停：

- 空闲/工作暂停 → 启动工作倒计时
- 工作运行中 → 暂停工作
- 工作完成/休息暂停 → 启动休息倒计时
- 休息运行中 → 暂停休息

按下 **KEY_ADD** (P3.4) 在设置模式下递加时间值：

- 设置工作时长模式时递加工作时长
- 设置休息时长模式时递加休息时长
- 范围 1~60 分钟，超过 60 自动回绕至 1

### 数码管显示

```
显示格式: MM.SS (分钟十位 分钟个位 . 秒数十位 秒数个位)

运行时:   分钟与秒之间的小数点以 0.5s 周期闪烁 (分秒分隔符)
暂停时:   小数点常亮，倒计时冻结
设置时:   正在调节的时间位以 0.5s 周期闪烁
空闲时:   显示预设时间，小数点常亮
```

### 工作流示例

一、上电后数码管显示 `2500` (默认工作 25 分钟)，小数点常亮，处于空闲模式

二、按 KEY_START_STOP → 开始 25 分钟工作倒计时，小数点闪烁

三、倒计时归零 → 蜂鸣器 5 声短鸣 → 自动切换至休息准备状态

四、按 KEY_START_STOP → 开始 5 分钟休息倒计时

五、休息归零 → 蜂鸣器 5 声短鸣 → 自动返回空闲，准备下一轮

如需调整时长，在空闲/运行/暂停状态下按 KEY_SET 进入设置模式，用 KEY_ADD 调节值，再按 KEY_SET 退出。

### 深度休眠与一键唤醒

系统在空闲/暂停状态下连续 5 分钟无操作后，自动进入 **STC89C52RC 掉电模式** (Power-Down Mode)，实现极致省电：

- **休眠条件**：系统处于 IDLE (空闲)、WORK_PAUSE (工作暂停)、WORK_DONE (工作完成) 或 REST_PAUSE (休息暂停) 状态，且 5 分钟内无任何按键操作
- **休眠动作**：关闭数码管段码与位选 (P0=0x00, P2|=0x0F)、停止 Timer0、配置 INT0/INT1 为下降沿触发 → `PCON |= 0x02` 进入掉电
- **休眠功耗**：晶振停振，全部外设停止，典型待机电流 **< 0.1 μA** (近乎零功耗)
- **唤醒方式**：按下 **KEY_SET** (P3.2, INT0) 或 **KEY_START_STOP** (P3.3, INT1) 瞬间唤醒
- **唤醒恢复**：自动重新初始化 Timer0、恢复数码管显示、重置 5 分钟计时器、消耗唤醒按键 (避免误触发功能)
- **运行中不休眠**：工作/休息倒计时运行中或设置模式中，即使无操作也不休眠

```
休眠触发条件:
  idle_seconds >= 300 (5分钟) 且 mode ∈ {0, 2, 3, 5}

掉电模式状态:
  PCON.PD = 1 → 振荡器停振
  INT0 (P3.2) 或 INT1 (P3.3) 下降沿 → 唤醒
  唤醒后从 PCON |= 0x02 的下一条指令继续执行
```

## 代码规范说明

本项目严格遵循参考文件 `E:/CLanguageGame/51Timer/main.c` 的技术规范，包含以下关键设计决策：

- 所有全局变量在 timer.c 中定义，在 config.h 中以 extern 声明，确保唯一定义点
- 数码管动态扫描全部在 Timer0 ISR 中完成 (主循环零负担)
- 按键消抖采用 10ms 阻塞复检策略，按下后等待释放防止连发
- 蜂鸣器告警为阻塞式，依赖 ISR 维持数码管刷新
- 模式系统使用 0~7 整数枚举而非复杂状态机，简化逻辑判断
- 休眠唤醒采用边沿触发 (IT0/IT1=1)，ISR 硬件自动清除中断标志，避免重入
- 掉电模式进入前关闭数码管和 Timer0，唤醒后自动恢复，主循环透明衔接
